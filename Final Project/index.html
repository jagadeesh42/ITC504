Inventory Management Dashboard

About This Dashboard

This web-based Inventory Management System allows users to manage products efficiently. You can:

  • Add, Edit, and Delete products.
  • Track stock levels and get alerts for low stock items.
  • Search and filter products easily.
  • Export inventory data to CSV for reporting.
  • View products in both table and card formats for convenience.

This dashboard is designed to provide a clean, professional interface suitable for IR and enterprise environments.
